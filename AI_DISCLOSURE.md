# AI Tooling Disclosure

Argus by Perciqa was developed with AI-assisted coding tools as part of the AMD Developer Hackathon Act II. We are disclosing this in the spirit of transparency and open-source integrity.

## Tools Used

| Tool | Purpose |
|---|---|
| **Google Antigravity (AGY)** | Primary AI coding assistant — used for code generation, architecture guidance, scaffolding, and pair programming throughout development |
| **Claude (Anthropic)** | Underlying model powering Antigravity sessions |

## Scope of AI Assistance

- **Architecture & design**: The product vision, PRD, and engineering design document were authored by the developer (human). AI was used to validate and refine ideas.
- **Code generation**: AI assisted in generating boilerplate, scaffolding, and implementation of components designed by the developer.
- **Code review**: AI was used to review generated code for correctness and consistency with the design.
- **All code was reviewed** by the developer before committing. No code was blindly staged.

## Human Contributions

- Product vision and market positioning
- Architecture and data model design
- All technical decision-making
- Prompt engineering and agent demo design
- Final review of all committed code

## Policy Compliance

This disclosure is made in accordance with the hackathon's guidelines on AI tooling. All AI-assisted contributions are covered under the MIT license applied to this repository.
